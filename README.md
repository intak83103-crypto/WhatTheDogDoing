# WhatTheDogDoing

我的刀盾！

計算機概論期末專題：文字互動式刀盾養成與戰鬥遊戲。

玩家可以建立使用者、培養自己的刀盾、購買與使用道具，並透過戰鬥測試挑戰不同敵人。遊戲目前以終端機文字互動為主，玩家可在不同介面輸入指令操作。

## 目前功能

- 使用者建立、切換、刪除與改名
- 刀盾建立、切換、改名與資訊查看
- 商店系統與購買確認流程
- 金幣扣款與背包系統
- 道具使用與指定刀盾目標
- 刀盾等級、經驗值與升級數值表
- 隨機元素屬性
- 戰鬥測試模式
- 敵人系統：史萊姆、哥布林、吸血鬼
- 技能資料表系統
- 多效果技能，例如傷害後吸血
- 技能冷卻時間
- Buff / Debuff 系統
- 可指定 Buff 目標為自己或對方
- 戰鬥中可查看自己與敵人的狀態、技能與目前 Buff

## 戰鬥系統

戰鬥使用速度累積行動值的方式決定出手順序。速度較高的角色可以更快行動，未來也能支援更明顯的速度差距。

目前技能支援：

- 攻擊
- 回復
- Buff
- 多段效果
- 命中後才觸發的後續效果
- 依攻擊力、最大生命值、目前生命值、已損失生命值計算效果
- 依造成傷害比例回復生命

Buff 可以影響：

- 攻擊力
- 最大生命值
- 速度
- 增傷
- 防禦
- 命中率
- 閃避率
- 爆擊率

Buff 也可以用負數表示 Debuff，例如降低防禦或降低速度。

## 操作提示

遊戲內多數介面可輸入：

```text
h / help
```

查看目前可用指令。

戰鬥中常用指令：

```text
1 ~ 4     發動技能
l / list  查看技能與 Buff
i / info  查看刀盾狀態
e / enemy 查看敵人狀態
q / quit  脫離戰鬥
```

## Build

本專案已拆分成 `include/` 與 `src/` 多檔案架構，使用 `Makefile` 編譯。

```bash
make
```

編譯完成後會產生執行檔：

```text
main
```

如果想清除編譯產物：

```bash
make clean
```

## Run

```bash
./main
```

## Project Structure

```text
main.cpp          程式入口
Makefile          編譯設定
include/          class 宣告檔
src/              class 實作檔
```

請在專案根目錄執行 `make` 與 `./main`。
